Getting Started

Welcome to Sanistar

  • This guide walks you through your first time using the system.
  • Everything here is focused on day‑one essentials — nothing deep or technical.
  • Once you’re comfortable, the other sections in the sidebar expand on each workflow.

What You’ll Learn

  • How to sign in
  • How to navigate the dashboard
  • How live inventory works
  • How to start and complete a build
  • How shortages and replenishment appear

Signing In

Use your assigned username and password. If you cannot sign in, your supervisor must activate your account.

Once logged in, you will land on the main dashboard.

Dashboard Overview

The dashboard shows:

  • Live Inventory — real‑time counts based on ledger activity
  • Production Queue — builds waiting to be started
  • Replenishment — items needing restock
  • Notifications — shortages, conflicts, or missing parts

Everything updates instantly — nothing is stored or stale.

Starting a Build Order

To begin a build:

  • Open the Production Queue
  • Select the build order
  • Review required parts and quantities
  • Tap Start Build

Sanistar immediately reserves the required parts using ledger entries.

Pulling Parts

When you physically pull parts from the shelf:

  • Tap Confirm Pull
  • Sanistar logs the consumption
  • Inventory updates instantly

If inventory is short, you’ll see a warning before you start.

Completing a Build

When the build is finished:

  • Tap Complete Build
  • Enter the quantity produced
  • Add optional notes

Sanistar automatically creates a replenishment entry for the finished goods.

Returning Parts

If you pulled parts but didn’t use them:

  • Open the build order
  • Tap Return Parts
  • Enter the quantity being returned

Sanistar logs a replenishment entry and releases the reserved demand.

Shortages

If inventory is insufficient, Sanistar will:

  • Flag the shortage on the dashboard
  • Send a deficiency notification
  • Prevent the build from starting until resolved
